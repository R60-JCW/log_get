# 削除したい列と行を指定する
# 列の削除の記載方法は、templateファイルで定義したValueのみを記載
# 行の削除の記載方法は、templateファイルで定義したValue名と削除行を一意に特定できる値を"&"でつなげて記載
# 列の記載例：CAPACITY    削除したい列のラベル名がCAPACITYの場合
# 行の記載例：ID&1        対応する列のラベル名がIDで、削除したい行の値が1の場合
# 行頭に"#"がある行は無視される
# 1行に列もしくは行が１つのみ(複数ある場合は改行する)
# 記載順不問(削除対象の行の記載と、削除対象の行の記載に要件はない)
# ただし、削除実行は列の要素から行うため、削除予定の列を行削除時のラベル名に指定するとエラーとなる。

E_MAJOR_THRESHOLD&110
ID&1
E_ID&1
E_ID&2
CAPACITY
VOLTAGE_SOURCE
VOLTAGE
